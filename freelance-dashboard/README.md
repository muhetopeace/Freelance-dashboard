# Freelance Dashboard (React + TypeScript)

## Description
A mini-dashboard to manage clients, projects and payments with TypeScript models and Context + useReducer for state.

## Technologies
- React + TypeScript
- Context API + useReducer
- Tailwind CSS (optional)

## Features
- Client list (optional emails)
- Project list with status & payment state
- Mark unpaid projects as paid
- Payment records and validation
- Utility functions: search, filter, counting paid/unpaid
- Type-safe models and discriminated union actions

## Setup
```bash
git clone <https://github.com/muhetopeace/Freelance-dashboard>
cd freelance-dashboard
npm install
npm run dev
